PATTERN ANALYSIS OF DISAGREEMENTS
================================================================================

PATTERN 1: TEMPORAL SUPERSESSION IN DISPUTE BATCH (5 of 13 disagreements)
  Cases: 548, 549, 551, 553, 556 (all in temporal_conflict subcategory of batch 4)

  Issue: These cases follow an identical pattern to the "contradiction_resolved"
  cases in batch 1 (where older sources are superseded by newer ones, and the label
  is "confident"). In the dispute batch, the same pattern (older source explicitly
  replaced by newer source) is labeled "disputed."

  The key distinction SHOULD be: if the newer source EXPLICITLY replaces the older
  one and the current state is clear, it's "confident." If both sources describe
  overlapping time periods with genuine uncertainty about which is correct, it's
  "disputed."

  In all 5 flagged cases, the newer source explicitly supersedes the older one:
    - API endpoint removed in newer version
    - Emergency bulletin restricts bridge load
    - Release notes drop Python 3.8 support
    - Operations notice converts meeting rooms
    - Rate update changes APY

  Recommendation: Relabel these 5 cases as "confident" OR move them to a
  "temporal_superseded" subcategory under the confident category.

PATTERN 2: DIFFERENT METRICS LABELED AS DISPUTES (3 of 13 disagreements)
  Cases: 510, 515, 522 (in implicit_contradiction subcategory of batch 3)

  Issue: These cases present two sources measuring genuinely different things
  (unemployment vs job-seeking, badge-in vs occupancy, displaced vs requesting
  assistance) and label the difference as a "dispute." A well-constructed dispute
  case should have sources making competing claims about the SAME factual question.

  Recommendation: Either relabel as "qualified" (sources provide partial/non-comparable
  data) or revise the contexts so both sources explicitly claim to measure the same
  thing.

PATTERN 3: BORDERLINE CONFIDENT/QUALIFIED (3 of 13 disagreements)
  Cases: 828, 844, 847 (in confident batch 1)

  Issue: These cases involve a high-quality dissenting source (Cochrane review,
  industry-standard report) that is dismissed via methodological critique. While
  the dismissal is defensible, a strict reader would see genuine tension.

  Recommendation: These are acceptable as-is for a TRAINING set (the classifier
  should learn to resolve these toward confident), but they should be acknowledged
  as borderline in evaluation metrics.
